r은 2의 2승, w는 2의 1승, x는 2의 0승을 의미함

4, 2, 1 각각의 합은 7

<br/>

chmod 770 ./Mino....

r은 사용자, w는 그룹 x는 제3자를 의미함

770은 사용자와 그룹한테 모든 권한 주겠다는 뜻

<br/>

777은 사용자, 그룹, 제3자한테 모든 권한을 준다는 뜻

<br/>

chmod 400 ...pem

읽기권한만 잊는 것

사용자만 읽을 수 있고 나머지는 권한이 없다는 뜻

<br/>

644는 사용자에게 read, write 가능

그룹은 read만
